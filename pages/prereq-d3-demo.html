Prerequisite map, D3-driven

Test page — not linked from Classes

Every course page's .prereq-map so far has been hand-typed HTML per course. This page instead keeps prerequisite data in a plain JS object (see moduleData in this page's source) and has D3 walk it and build the exact same .prereq-station/.prereq-dot/.prereq-label/.prereq-branch markup from prereq-branch-themed.html — so city theme and dark mode keep working for free, same CSS, zero duplication. The map is also click-through: any grayed station is a real prerequisite of the one currently shown, click it to re-center the map on that module and see its chain.
