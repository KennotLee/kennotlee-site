Prerequisite Branch Style Gallery

Test page — not linked from Classes

Eleven different ways to draw an OR-branch (EE1111A OR EE2111A) merging into a line, all showing the same example so they're directly comparable. Pick whichever one actually looks good and tell me the number — or copy/adjust the CSS yourself (each one is a self-contained block below, search stylesheets/style.css for "GALLERY" to find the ones that aren't already part of the real .prereq-branch component used on classes/ee2028.html).

12–17 take the "eased elbow" family (9, 10, 11) further, two ways: 12–14 mirror the same branch shape onto the OUTPUT side too (this course unlocking either of two follow-on courses, not just being unlocked by one); 15–17 recolor 9/10/11 with the same solid-gray "already done" treatment the real course pages use for prerequisites, so everything is grayed except the final (current) course. All fictional course codes, same as 1–11.
