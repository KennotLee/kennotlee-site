Themed Branch Style Gallery (9–17)

Test page — not linked from Classes

Same nine shapes as entries 9–17 in prereq-styles.html (the "eased elbow" family: plain, branching on both sides, and grayed), but rebuilt to actually respond to the theme — every station below is the real .prereq-station/.prereq-dot/.prereq-label component, not a fixed SVG circle, so switching the footer's city theme changes the dot shape and size (BART tick, NYC ring, Tokyo badge...) live, and the connector lines' thickness and color follow --prereq-line-height/--accent/--border the same way the real map does. Only the curved connectors themselves are still one-off SVG (no CSS equivalent for this particular bend shape exists yet). Boxed stations are real links (href="#" placeholders) and lift on hover, same "hover only if there's a link" rule as everywhere else. GALLERY-ONLY markup — see the "GALLERY DIAGRAM, THEMED" comment in stylesheets/style.css for exactly how the positioning math works.
