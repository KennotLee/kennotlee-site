CS2040 — Data Structures and Algorithms

Computing Second Major Semester 2 AY25/26

Incomplete content — this page is incomplete

This class is a classic Data Structures and Algorithms course. Lectures are done online via zoom and recorded attendance optional. Tutorials have class participation and attendance taken. Goes through the tutorials for each week. Labs are just submitting a pseudocode of the kattis assignments and then coding on Java afterwards. You just need to submit the pseudocode and then you can code on your own time.

Course contents:

  1. Time Complexity
  2. Sorting Algorithms
  3. Linked Lists
  4. ArrayList, Linked List
  5. Stack, Queue
  6. HashMap/Set
  7. Priority Queue/Heap
  8. Union Find/Disjoint Set
  9. Binary Search Tree/AVL Tree
  10. Graphs
  11. Minimum spanning tree
  12. single source shortest path
  13. all pairs shortest path

Grades breakdown:

  • In Lab Assignments (ILA): 10%
  • Midterm: 30%
  • Final Exam: 50%

Took this class over CS2040DE because I wanted to take it with my friends tbh. Also I did not really entrust a CS module to be handled by ECE department since I have heard feedback that the admin side of the module was not well run. I also wanted to challenge myself and just not coop myself up in Engineering. This module is probably harder than the DE variant. DE variant is filled with ISE Majors and Engineering Majors with Computing (Design and Engineering) Minor / 2nd Major while this was filled with DSA and BZA students. When comparing both variants I preferred the style done by SOC since it's more structured and the questions were reasonably challenging. (Or maybe I'm biased by my choices)
